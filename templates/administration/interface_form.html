Interface Access Control Help
What are Interfaces?

Interfaces represent different parts of the system that users can access. Each interface controls access to specific functionality or pages.

How Access Control Works

Once an interface is created, you can assign specific users access to it. Users will only be able to access interfaces they have been granted permission to use.
